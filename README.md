# Utility Meter Dashboard

A simple web application that allows users to track electricity or water meter readings and calculate usage.

This project was built using Node.js and Express as a beginner full-stack developer project.

The system allows users to add meter readings and automatically calculates usage based on previous and current readings.

---

## Features

• Add new meter readings  
• Automatically calculate usage  
• View stored meter records  
• Simple web dashboard interface

## How It Works

The user enters:

Meter ID  
Previous Reading  
Current Reading  

The system calculates:

Usage = Current Reading − Previous Reading

Example:

Meter ID: 1001  
Previous: 4500  
Current: 4700  

Usage: 200 units

## Installation

1. Clone the repository
git clone https://github.com/dumisanimabuza087-svg/utility-meter-dashboard.git

2. Navigate into the project folder
cd utility-meter-dashboard

3. Install dependencies
npm install

4. Start the server
node server.js

5. Open browser
http://localhost:3000
